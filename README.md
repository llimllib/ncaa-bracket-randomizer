# NCAA Bracket Randomizer

Generate a random bracket that's not completely outrageous, for use in your office pool, or just to mess around with.

- randomize until your team wins it all!
- slider to go from chalk (favorite wins every game) to chaos (every game is 50/50)
- it's cheap! Only zero dollars

![screenshot of a randomized bracket](docs/screenshot.png)

Uses [Torvik probabilities](https://barttorvik.com/) (csv [version here](https://barttorvik.com/2026_team_results.csv))
